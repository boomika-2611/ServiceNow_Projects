**Automated Network Request Management – ServiceNow**

**Overview**

This project automates the end-to-end lifecycle of network service requests using ServiceNow. It enables users to submit network requests through the Service Portal and automates approval, task creation, data storage, and notifications using Flow Designer.

**Key Features**

Service Catalog item for network requests

Dynamic forms using UI Policies

Automated approvals using Flow Designer

Automatic task creation for network team

Email notifications at each stage

Custom Network Database table for tracking requests

**Modules Used**

Service Catalog

Flow Designer

Custom Tables (u_network_database)

UI Policies & Business Rules

Email Notifications

**Workflow Summary**

Service Portal → Network Request → Submit → Approval → Task Creation → Notification → Closure

**Testing**

The solution was tested for:

Catalog submission

Approval flow

Task creation

Flow completion

Email notifications

**Conclusion**

This project improves operational efficiency by reducing manual effort, enforcing standardized workflows, and providing better visibility into network request handling.
